# Troubleshooting

- **Not enough ratings:** Add more (≥10) to unlock behavioral recs.  
- **Search looks odd:** Title searches use **title similarity** ranking; subject searches use **Bayesian popularity**.  
- **Profile not updating:** Use the **Edit Profile** button and refresh.  
- **Missing metadata:** Some entries are incomplete, but enrichment is planned.  
- **Chatbot lost context:** Check that cookies are enabled; memory may also expire automatically.
