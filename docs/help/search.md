# Search Page

- **Default view:** Books are sorted by **Bayesian popularity** (average rating adjusted for rating counts).  
- **Search by subject:** Results remain **Bayesian-ranked** within that subject.  
- **Search by title:** Results are ranked by **title similarity**. You can also filter these results further by **subject**.  

- Results show book cards with the book cover retrieved from Open Library, title, author, year and ISBN.
- Clicking on a book card will redirect users to the book's page where there will be more info.
- It is not possible to search by author at the moment.

- Suject filtering is an AND operation so all subjects must match. 
- If users want a more robust way of searching by subjects they can select/change their favorite subjects on their profile page and use the subjects recommendations, adjusting the popularity bias as they like to find books that align with the chosen subjects.
